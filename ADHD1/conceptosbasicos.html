Conceptos básicos

Un sitio web es una colección de páginas web relacionadas entre sí y que son accesibles a través de Internet.Un sitio web puede tener diferentes propósitos, como proporcionar información, vender productos o servicios, entretener, educar, o facilitar la interacción y comunicación entre usuarios.

Una página web es un documento digital que forma parte de un sitio web y puede ser visualizado a través de un navegador web.Cada página web tiene una dirección única llamada URL Cuando un usuario ingresa la URL de una página web en un navegador, el servidor web entrega el código HTML correspondiente al navegador del usuario, y este último interpreta y muestra la página web.
Las páginas web pueden contener texto, imágenes, videos, enlaces, formularios y otros elementos multimedia. Además, las páginas web pueden ser estáticas, con contenido fijo, o dinámicas, con contenido que puede cambiar según la interacción del usuario o la información proveniente de una base de datos. Esta ubicación puede ser otra página en el mismo sitio web, una página en un sitio web diferente, un archivo descargable, o incluso una ubicación específica dentro de la misma página (enlaces anclados).

Es un elemento en un documento digital que permite a los usuarios navegar de una ubicación a otra dentro del mismo documento o a documentos diferentes. Esta ubicación puede ser otra página en el mismo sitio web, una página en un sitio web diferente, un archivo descargable, o incluso una ubicación específica dentro de la misma página (enlaces anclados).

Se refiere a la dirección única que identifica un recurso en la web. Una URL es la forma estándar de referenciar y acceder a recursos en Internet, como páginas web, imágenes, archivos, servicios web, entre otros. Una URL generalmente se compone de varios componentes: Protocolo, dominio, ruta y parámetros.

Es un estándar utilizado para la creación y el diseño de páginas web. Es un lenguaje de marcado que proporciona la estructura del sitio web.

CSS se utiliza para dar estilo y formato a los elementos HTML en una página web, permitiendo a los desarrolladores web separar la estructura del contenido de su presentación visual. Las reglas de estilo en CSS se aplican a elementos HTML mediante la asignación de propiedades y valores a dichos elementos. Esto incluye el control de aspectos como el color, la fuente, el espaciado, el tamaño, la disposición y otros aspectos visuales de la página.

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se utiliza principalmente en el desarrollo web. Es uno de los tres componentes fundamentales de las tecnologías web junto con HTML y CSS.JavaScript permite crear contenido web dinámico e interactivo, lo que significa que las páginas web pueden responder a las acciones del usuario sin necesidad de recargar la página.

El DNS actúa como una especie de directorio que permite a los usuarios acceder a sitios web utilizando nombres de dominio en lugar de tener que recordar direcciones IP numéricas.

La dirección IP es esencialmente un identificador único para un dispositivo en una red, ya sea una computadora, un servidor, un enrutador u otro dispositivo conectado. Cuando envías datos a través de Internet, estos se dividen en paquetes, y cada paquete contiene la dirección IP de origen y destino. Los enrutadores utilizan esta información para dirigir los paquetes correctamente a través de la red hasta su destino. En resumen, el Protocolo de Internet (IP) es un conjunto de reglas que permite la comunicación entre dispositivos en redes, y las direcciones IP son los identificadores numéricos asignados a esos dispositivos para facilitar la correcta transmisión de datos en la red.

Un dominio web es una identificación alfanumérica fácil de recordar que se utiliza para acceder a recursos en Internet, como sitios web. En lugar de utilizar direcciones IP numéricas, que son identificadores únicos para cada dispositivo en una red, los dominios web permiten a los usuarios acceder a sitios de una manera más amigable y legible.

La relación cliente-servidor es un modelo de arquitectura en el que dos componentes, llamados cliente y servidor, interactúan entre sí para realizar operaciones y compartir recursos en una red. Este modelo es fundamental en la programación distribuida y en sistemas de comunicación en red

Es un protocolo estándar utilizado para la transferencia de archivos entre un cliente y un servidor en una red, como Internet. FTP permite a los usuarios cargar (subir) y descargar (bajar) archivos desde o hacia sus servidores. Es importante mencionar que, aunque FTP es un protocolo ampliamente utilizado, la transferencia de archivos a través de FTP no está cifrada y la información, incluyendo las credenciales de inicio de sesión, se transmite en texto claro.

es una versión segura del protocolo HTTP utilizado para la transmisión segura de datos en la World Wide Web. HTTPS añade una capa adicional de seguridad al cifrar la información que se transmite entre el cliente (como un navegador web) y el servidor web. Este cifrado se realiza mediante protocolos de seguridad como SSL (Secure Sockets Layer) o su sucesor TLS (Transport Layer Security).

El término "frontend" se refiere a la parte de un sistema informático, aplicación web o software que interactúa directamente con los usuarios y se encarga de la presentación y la interfaz gráfica. En el desarrollo de software, el frontend es la capa visible y accesible para los usuarios finales. Incluye todos los elementos con los que los usuarios pueden interactuar, como la interfaz de usuario, los botones, los formularios y otros elementos visuales.

El término "backend" se refiere a la parte del sistema informático o de una aplicación que está detrás de escena y se encarga del procesamiento, la lógica de negocios, la gestión de datos y la comunicación con el frontend. En otras palabras, el backend es la capa que maneja la funcionalidad que no es directamente visible para los usuarios finales, pero que es esencial para el funcionamiento de la aplicación.

se refiere al servicio que proporciona la infraestructura necesaria para almacenar y hacer accesibles los archivos y datos de un sitio web en Internet. Cuando tienes un sitio web, necesitas un lugar para almacenar sus archivos, imágenes, contenido y otros elementos relacionados. Ese lugar es proporcionado por un proveedor de servicios de hosting.

es un tipo de servicio de alojamiento web que proporciona una porción virtualizada de un servidor físico. Aunque varios clientes comparten el mismo servidor físico, cada uno tiene asignados recursos específicos, lo que le otorga características similares a las de un servidor dedicado.

El hosting en la nube, también conocido como alojamiento en la nube o cloud hosting en inglés, es un tipo de servicio de alojamiento web que utiliza recursos de varios servidores interconectados para proporcionar almacenamiento y capacidad informática. En lugar de depender de un único servidor físico, como en el hosting tradicional, el hosting en la nube aprovecha la infraestructura distribuida de servidores para mejorar la escalabilidad, la disponibilidad y la flexibilidad.

W3C, que significa "World Wide Web Consortium" (Consorcio World Wide Web), es una organización internacional que se dedica al desarrollo de estándares y protocolos para la World Wide Web. Fue fundado en 1994 por el creador de la web, Sir Tim Berners-Lee, y es dirigido por un equipo de expertos de diversas organizaciones y empresas de tecnología. El objetivo principal del W3C es promover la interoperabilidad y la evolución coherente de la World Wide Web. Para lograr esto, el consorcio trabaja en la creación y mantenimiento de estándares abiertos y tecnologías web.
